Image Assets for Regiment Cards
===============================

Place your .png or .jpg images in this folder.
They must be set as "Sprite (2D and UI)" in Unity's Import Settings.

Required File Names:
--------------------
1. Card Frame:
   - "CardFrame"

2. Unit Icons:
   - "Icon_LineInfantry"
   - "Icon_LightInfantry"
   - "Icon_Cavalry"
   - "Icon_Artillery"

If these files are missing, the game will use the default code-generated look.
